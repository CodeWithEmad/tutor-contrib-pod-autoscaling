pod-autoscaling plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/eduNEXT/tutor-contrib-pod-autoscaling

Usage
-----

::

    tutor plugins enable pod-autoscaling
